## Codebase Patterns
- Doge Code is a Bun-first TypeScript CLI project with the main entrypoint at `src/bootstrap-entry.ts`.
- Root-level validation is command-driven: prefer `bun run version` for a boot smoke test and targeted manual checks for the specific command or flow you change.
- This restored source tree values minimal, auditable changes over broad refactors.
- `process.versions.bun` gives the Bun version synchronously; `process.version` gives Node version.
- Doctor screen uses React compiler (`_c(n)` cache sentinel pattern) — small JSX inserts work fine but avoid restructuring the cache indices.
